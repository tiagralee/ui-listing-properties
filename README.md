## Property Listing Project

## Environment:
- [Node.js](https://nodejs.org/en/)
- [npm](https://www.npmjs.com)

## Run the project:
- cd to the project directory
- Run `npm install`
- Run `npm start`
- Go to `http://localhost:3000`

## Build: 
- Run `npm build`
- See the build app in `/build` directory

## Run test:
- Run `npm test`
